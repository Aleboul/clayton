Welcome to clayton's documentation!
===================================

The module clayton is a object-oriented of the Python language module to sample 
random numbers from copulae. For more details about copulae and the module, check
out the `ArXiv https://arxiv.org/abs/2203.17177` paper.

Check out the :doc:`usage` section for further information, including how to
:ref:`install <installation>` the project.
